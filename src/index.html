How to use

Welcome to the Tourism New Zealand Journey Planner. This APP allows you to create a custom journey through NZ, and will recommend a rental vehicle to suit your journey.

1:  Click to start

2:  Enter the start and end location for your journey

3:  Enter the number of days you need the vehicle for

4:  Enter the number of people travelling

5:  Select the vehicle from the options provided
